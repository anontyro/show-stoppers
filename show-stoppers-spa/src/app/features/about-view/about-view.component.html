Show Stoppers

Show Stoppers is about all the show stopping TV shows that are found internationally. Show stoppers gets all its information from The Movie Db and displays them here to be viewed. Each show is broken down into their seasons and episode to make it easier to find what you are after

The Stack

This site uses a number of technologies to make everything possible and talks extensively to The Movie DB API to get all the data here is the list:

  • The API - Hosting on AWS Lambda
  • The Frontend - Hosted on my site which is running on Digital Ocean the frontend section is written in Angular 6
